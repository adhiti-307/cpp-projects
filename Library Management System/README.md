# Library Management System (C++)

This is a simple console-based Library Management System built using C++.  
It allows users to add books, search books, view available books, and manage library records.

## Features
- Add new books to the library
- Display all available books
- Search for a book by title
- Search for a book by author
- Search for a book by ISBN
- Basic menu-driven interface


## Technologies Used
- C++
- Standard Template Library (STL)

## Future Improvements
- Member registration & issue/return system
- Store data permanently using files or database
- GUI version
